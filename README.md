# F-Guard KE

F-Guard KE is a **forest monitoring and alerts dashboard** built with **React.js** and **plain CSS**.  
The app allows users to view real-time forest alerts, monitor hotspots, track reports, and manage settings. Designed for quick prototyping in hackathons.

---

## Features

- **Sidebar navigation** with icons  
- **Home Dashboard** with alert cards and stats  
- **Pages:** Home, Live Map, Alerts, Reports, Settings  
- **Responsive cards** for alerts and metrics  

---

## Project Structure
  src/
├─ components/ ← Reusable components like Sidebar, Card
├─ pages/ ← Page components (Home, Alerts, Reports, Settings)
├─ App.jsx ← Main app layout
├─ main.jsx ← ReactDOM entry
└─ style.css ← All styling (plain CSS)


---

## Installation

1. Clone the repository:

bash
git clone <your-repo-url>
cd f-guard-vite

Install dependencies:

npm install


Start the development server:

npm run dev


Open http://localhost:5173
 in your browser.
